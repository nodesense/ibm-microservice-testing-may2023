
Open New Terminal


```
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```

close the terminal

open new terminal [since path is updated on new terminal]

install node.js 16

```
nvm install 16
```

check node version

```
node -v
```
